Working with Components

Components allow you to re-use content across pages in your site. Creating a component is as easy as creating a page. And, you use the same visual editor to make updates, so all your users can edit components with ease.

You can place components in pages using the page editor. Simply drag the component onto the page and you will see the content appear on that page.

Build fast, responsive sites with Respond.

Built by Matthew Smith in Manchester, MO. Respond CMS is the white-label, flat-file, multi-site, incredibly fast CMS of your dreams built on the Lumen PHP and Angular frameworks. Respond is built so that you can quickly build beautiful, responsive websites. The latest version features flat-file storage, a beautiful visual editor, and fast static sites.
