Acknowledgements

This is just a quick shout out to all the great open source projects that are used to make Respond great.

  • Font Awesome - A lot of the icons used within the editor come from Font Awesome.  This project continues to make the lives of developers easier.
  • Magnific Popup - Magnific powers the lightbox feature of Respond.
  • AngularJS - The fantastic front-end framework that powers the app and sites in Respond.
  • Bootstrap - Powers the layout and style of the apps and sites.
  • Unsplash - The ultra cool resource for beautiful, free stock photos.  They make our themes look great.
  • LessPHP - The library that powers our LESS to CSS conversions.
  • TonicPHP - The framework that powers our REST API.
  • Guzzle - Guzzle provides the backend for our Amazon implementation, as well as, the framework for our webhooks implementation.
  • PHPMailer - Provides the underlying framework for emails sent in the system.
  • Composer - Handles dependency management within the application.
  • Firebase PHP-JWT - Provides our web token authentication implementation.
  • jQuery - Used to simplify some of our scripting.
  • i18next - Used for internationalizing the app and the sites.
  • ng-i18next - Integrates the i18next JS framework with AngularJS
  • CodeMirror - Used for editing HTML and CSS within the application.
  • MomentJS - Handles time within the application

A big thank you to all those who build these great projects.

Build fast, responsive sites with Respond.

Built by Matthew Smith in Manchester, MO. Respond CMS is the white-label, flat-file, multi-site, incredibly fast CMS of your dreams built on the Lumen PHP and Angular frameworks. Respond is built so that you can quickly build beautiful, responsive websites. The latest version features flat-file storage, a beautiful visual editor, and fast static sites.
